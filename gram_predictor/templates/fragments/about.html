About This System

Project Introduction

The Gram-Negative Bacteria Prediction System is an online tool based on advanced machine learning techniques, designed to predict whether peptide sequences exhibit activity against Gram-negative bacteria. This system employs an innovative hybrid model combining deep learning and traditional bioinformatics features to effectively capture the structural and functional properties of peptide sequences.

Core Technologies
  • Hybrid Prediction Model: A Keras-based model that integrates a Long Short-Term Memory (LSTM) network for sequence analysis with a Multi-Layer Perceptron (MLP) for physicochemical feature analysis.
  • Advanced Sequence Generation: A D3PM (Discrete Denoising Diffusion Probabilistic Model) powered by ESM-2 embeddings to generate novel, high-potential antimicrobial peptide sequences.
  • Interactive Web Interface: A dynamic, single-page application experience built with Flask, Bootstrap, and JavaScript for seamless user interaction.
Usage Instructions
  1. Predict: Navigate to the Predict section. Upload a FASTA file or paste sequences directly. The system will return predictions with probabilities and detailed statistics.
  2. Generate: Go to the Generate section. Configure parameters like sequence length and sampling method to create novel peptide sequences.
  3. Predict Generated Sequences: After generating sequences, use the "Predict Activity" button to instantly analyze their potential without leaving the page.
