About This System

Project Introduction

The Gram-Negative Bacteria Prediction System is an online tool based on advanced machine learning techniques, designed to predict whether peptide sequences exhibit activity against Gram-negative bacteria. This system employs an innovative hyperbolic space embedding model, capable of effectively capturing the structural features and functional relationships of peptide sequences.

Technical Principles

This system is based on the following technologies and principles:

  • Hyperbolic Space Embedding: Utilizes the Poincaré ball model for hyperbolic space embedding, which can better represent the hierarchical structure of biological sequences.
  • Attention Mechanism: Combines attention mechanisms with hyperbolic distance metrics to enhance the model's expressive power.
  • Multi-Prototype Representation: Adopts a multi-prototype/cone point design to improve the model's ability to capture complex features.
  • Peptide Feature Extraction: Calculates various physicochemical properties, including charge, hydrophobicity, amphipathicity, hydrophilicity, etc.
Usage Instructions

Using this system is very simple:

  1. On the homepage, select "File Upload" or "Text Input" to provide peptide sequences in FASTA format.
  2. Click the "Start Prediction" button, and the system will automatically extract features and make predictions.
  3. View the prediction results, including statistical information, charts, and a detailed prediction results table.
  4. You can choose to export the results as a CSV file or export positive sequences as a FASTA file.

Frequently Asked Questions

Anti-Gram-negative peptides are a class of short peptides that exhibit antimicrobial activity against Gram-negative bacteria (such as E. coli, Pseudomonas aeruginosa, etc.). These peptides typically possess specific physicochemical properties, such as amphipathic structures and positive charges, enabling them to disrupt bacterial cell membranes or interfere with essential bacterial physiological processes, thereby inhibiting or killing the bacteria.

Our system achieved an accuracy of 92.8% and an F1 score of 92.8% on an independent test set. However, prediction results should still be combined with experimental validation, especially for borderline cases where the prediction probability is close to 0.5.

To ensure system performance, a single prediction supports up to 20,000 sequences. If you need to process more sequences, it is recommended to submit them in batches or download our offline version.

The prediction results include a prediction probability and a prediction label. The prediction probability indicates the confidence level that a sequence is classified as anti-Gram-negative, ranging from 0 to 1. Sequences with a prediction probability ≥ 0.5 are labeled "Anti-Gram-Negative," otherwise they are labeled "Non-Anti-Gram-Negative." The closer the prediction probability is to 1, the more confident the model is that the sequence possesses anti-Gram-negative activity.

The system first extracts various physicochemical properties from the input peptide sequences, including charge, hydrophobicity, amphipathicity, amino acid composition, etc. Then, these features are fed into a trained hyperbolic space embedding model. The model maps the features to hyperbolic space and uses attention mechanisms and multi-prototype representations for classification. Finally, the model outputs the probability for each sequence to be classified as anti-Gram-negative.
